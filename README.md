# SpringBootNoviceProject
一个简单的图书项目用来熟习springboot技术，非常简单 是根据以前的的javaweb考试题目做的

### 开发环境
| 工具| 版本或描述|    
| ------- | ------------------------ |    
| OS | Windows 10 |
| `JDK` | 1.8 |    
| `IDE` | IntelliJ IDEA 2019.1版|    
| `Maven` | 3.6.0 |    
| `MySQL` | 8.0.16 |
| `bootstrap` | 3.2.1 |
