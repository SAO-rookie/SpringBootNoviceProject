# SpringBootNoviceProject

### 开发环境
| 工具| 版本或描述|    
| ------- | ------------------------ |    
| OS | Windows 10 |
| `JDK` | 1.8 |    
| `IDE` | IntelliJ IDEA 2019.1版|    
| `Maven` | 3.6.0 |    
| `MySQL` | 8.0.16 |
| `bootstrap` | 3.2.1 |
